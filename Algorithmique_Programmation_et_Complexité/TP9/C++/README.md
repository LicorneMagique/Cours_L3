## Lancer le programme

Le programme attend comme paramètres le nom du fichier image source (src.pgm), le nom du fichier à produire (dest.pgm), alpha et éventuellement sigma

```shell
make
# Sans sigma
./main image4.pgm toto 50

# Avec sigma
./main image4.pgm toto 50 127
make clean
```
